# Sketchfab Simple Car

Preferred source for the project car model:

- Model: Simple car
- Author: IanMasterGG
- Source: https://sketchfab.com/3d-models/simple-car-f33baaae7c444d239faf32200453f981
- License: Creative Commons Attribution 4.0
- License URL: http://creativecommons.org/licenses/by/4.0/

Sketchfab's download endpoint requires an authenticated account token from this
environment. Download the OBJ archive from the model page in a browser, extract
it, and place the OBJ at:

```text
assets/models/sketchfab-simple-car/simple-car.obj
```

The renderer checks this file first. If it is missing, the project falls back to
the bundled CC0 Kenney car model.
